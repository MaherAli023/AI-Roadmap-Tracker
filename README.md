# AI Roadmap Tracker

AI Roadmap Tracker is a full-stack web application that helps users explore and generate AI learning roadmaps for different technologies, domains, and career paths.

The project consists of a separate frontend and backend architecture, providing a scalable and modular setup for roadmap generation and management.

---

# Features

* AI roadmap generation
* Interactive roadmap visualization
* Category-based roadmap organization
* Responsive user interface
* Frontend and backend separation
* API-based architecture
* Modern UI components and icons

# Installation & Setup

## Clone Repository

```bash
git clone https://github.com/MaherAli023/AI-Roadmap-Tracker.git
```

## Move Into Project Folder

```bash
cd AI-Roadmap-Tracker
```

---

# Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

Frontend runs on:

```bash
http://localhost:3000
```

---

# Backend Setup

```bash
cd backend
npm install
npm start
```

Backend runs on:

```bash
http://localhost:5000
```
